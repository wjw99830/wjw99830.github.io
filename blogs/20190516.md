# 算法
看到一道题: 找出数组中唯一不重复的元素, 时间复杂度O(n), 空间复杂度O(1);
解为所有元素异或的结果, 即a1 ⊕ a2 ⊕ a3 ⊕ ..., 最后得到的值即不重复出现的值(证明过程不详);
